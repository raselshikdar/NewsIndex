# NewsIndex

**NewsIndex** is a web-based news aggregation project that brings together headlines and links from multiple Bangladeshi and international news sources in one place.

This project does **not publish original news content**. It acts only as an index and discovery platform, redirecting users to the original publishers for full articles.
